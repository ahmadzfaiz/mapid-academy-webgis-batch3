# mapid-academy-webgis-batch3

## Step 1 - Check NodeJS installation
```bash
node -v
npm -v
```

## Step 2 - Build a new Vite Project
```bash
npm create vite@latest
```

## Step 3 - Check Vite local development server
```bash
npm run dev
```

## Step 4 - Add MapLibre in the package
```bash
npm install maplibre-gl
```

## Step 5 - Add Bootstrap CSS
```bash
npm install bootstrap
```
